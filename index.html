TERMINAL COMMAND

  • Directories
$pwd
Display path of current directory
$cd
Change directory
$cd
Change directory
cd ..
Navigate to parent directory
ls
List directory contents
ls -la
List detailed directory contents,including hiden files
  • Files
$rm file
Delete file
$rm -r directory
Delete Directory
$rm -f file
Force-delete file (add -r to force-delete a directory)
$mv file-old file-new
Move file to directory (possible overwriting an existing file)
cp file directory
Copy file to directory (possible overwriting an existing file)
cp -r directory1 directory2
Copy directory1 and its contents to directory
$touch file
Update file access & modification time (create file it it doesn't exist)
